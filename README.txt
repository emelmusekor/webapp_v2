2026 KERIS AI 리터러시 검사도구 · 모듈1 (AI 실습 중심)
"함께 만드는 환영 앨범" — 학생용 검사 웹앱
============================================================

[개요]
해외 교류 친구를 위한 환영·추억 앨범을 AI 콘텐츠 제작 도구로 만드는 7단계 수행형 검사.
각 단계는 하나의 성취수행 예시만 측정하며, 실제 AI 도구 사용 경험을 재현한다.

[실행 방법]
- 단일 페이지 웹앱(SPA). 별도 빌드/설치 불필요.
- 로컬 서버에서 열기 (파일 직접 열기는 일부 브라우저에서 제한될 수 있음):
    python -m http.server 5191
  후 브라우저에서 http://localhost:5191/ 접속.

[파일 구조]
- index.html : SPA 셸 (사이드바 + 메인)
- style.css  : 디자인 토큰 및 컴포넌트 스타일
- app.js     : 라우터(해시), 진행상태 저장(localStorage), 메인 시나리오, 1~7단계 로직
- assets/svg : (선택) 보조 그래픽

[단계 구성]
1단계. 번역 결과 역번역으로 검증하기 (B2-3)
   - 구글 번역기형 UI에서 한국어 입력 → 최대 5회 번역 누적
   - 각 번역을 '한국어로 역번역'해 원문과 비교, 의미 보존된 번역 선택
   - 핵심: 번역기를 맹신하지 않고 역번역으로 직접 검증 (사용 전 역번역 확인 강제)
   - 정답: 5회 모두 번역 + 선택 번역 역번역 확인 + 의미 보존된 5번째 선택

2단계. 사진에 넣을 영어 문구 만들기 (B2-2)
   - AI 문구 후보 중 활동과 맞고 짧고 존중적인 문구 선택 (놀림/과장/고정관념 배제)

3단계. 사진 배경 정리하기 (B2-2)
   - 활동 장면은 남기고 지나가는 얼굴·불필요 배경만 제거 (정보 선별 편집)

4단계. 사진으로 추억 영상 만들기 (B2-2)
   - 길이/분위기/전환/구성을 목적(따뜻한 추억 영상)에 맞게 선택

5단계. 자동 번역 자막 확인하기 (B2-3)
   - AI 자동 번역 자막의 오역(이름·장소·낱말 의미)을 찾아 수정

6단계. 친구 사진 사용 동의 확인하기 (C2-3)
   - 미동의 얼굴 흐리게 + 개인정보 이름표 가리기

7단계. 공유 전 표현과 범위 점검하기 (C2-2)
   - 대상화 표현을 존중적 표현으로 수정 + 공유 범위(학급) + AI 생성 표시

[설계 원칙]
- 학생용 검사 화면과 교사용 안내를 분리: 교사용 정답/채점기준은 각 단계 하단의
  접힌 '교사용 정답/채점기준 보기'에만 둔다. 학생 화면에는 채점 힌트를 노출하지 않는다.
- 외국어 실력이 아니라 'AI 산출물을 검증·수정·개선하고 윤리를 지키는 수행'을 측정한다.
- 진행상태는 localStorage('ai_lit_m1_*')에 저장되며, 메인페이지에서 전체 초기화 가능.
